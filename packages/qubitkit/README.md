# QubitKit
Quantum circuit partitioning toolkit.

## Usage
```python
import qubitkit
qubitkit.hello()
```

## functionalities

Target: 10 qubits, 20-30 gates

### Essentials
1. Random Circuit Generator
2. Circuit File reader
